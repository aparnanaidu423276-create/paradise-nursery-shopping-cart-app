# paradise-nursery-shopping-cart-app
Paradise Nursery is a modern, user-friendly online plant shopping application built using React and Redux. The application allows users to browse a wide variety of houseplants, view plant details such as images, names, categories, and prices, and add selected plants to a shopping cart. 
Technologies Used:
1.React
2.Redux Toolkit
3.JavaScript (ES6)
4.HTML5
5.CSS3

Key Features:
Product listing with multiple plant categories
Add to Cart functionality with disabled state after adding
Dynamic cart item count and total price calculation
Increase, decrease, and remove cart items
Responsive navigation bar
Simple and clean user interface
