# Paradise Nursery Shopping Cart Application

Paradise Nursery is a React-based shopping cart application for an online plant store.
Users can browse plants by category, add items to a cart, update quantities, and view total costs.

## Features
- Browse plants by categories
- Add plants to shopping cart
- Increase or decrease item quantity
- Remove items from cart
- Dynamic cart count using Redux
- Clean and responsive UI

## Technologies Used
- React
- Redux Toolkit
- React Router
- JavaScript
- CSS

## Project Purpose
This project was built as part of a Coursera Final Project to demonstrate React and Redux concepts.
